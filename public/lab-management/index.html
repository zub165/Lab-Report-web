    CSV File Format Guide

    Your CSV file should have the following columns:

    • Test Name - Name of the test
    • Description - Test description
    • Price - Test price (numeric)
    • Parameters - Semicolon-separated parameter names
    • Units - Semicolon-separated units
    • Reference Ranges - Semicolon-separated ranges

    Sample CSV content:

    Test Name,Description,Price,Parameters,Units,Reference Ranges
Blood Test,Complete Blood Count,50.00,"Hemoglobin;RBC;WBC","g/dL;million/µL;K/µL","13-17;4.5-5.9;4.5-11.0"
Lipid Profile,Cholesterol Test,75.00,"Total;HDL;LDL","mg/dL;mg/dL;mg/dL","150-200;40-60;70-130"
